# Description
Ce projet présente une Analyse de données exploiratoire des images radio d'une portion du dataset du NHI sur les images radio du thorax suivi d'une implémentation du modele [CheXNet](https://arxiv.org/abs/1711.05225).

Le notebook original présent sur Kaggle a obtenu plus de 1000 vues. 
Lien : https://www.kaggle.com/code/roblexnana/assistance-radiologie-chexnet
